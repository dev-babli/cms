## ğŸ¯ How to Contribute

### 1. **Report Bugs**

Found a bug? [Open an issue](https://github.com/your-username/emscale-cms/issues)

Include:

- Description of the bug
- Steps to reproduce
- Expected vs actual behavior
- Screenshots if applicable

### 2. **Suggest Features**

Have an idea? [Open a feature request](https://github.com/your-username/emscale-cms/issues)

Include:

- Clear description
- Use case
- Proposed solution

### 3. **Submit Code**

```bash
# Create feature branch
git checkout -b feature/your-feature

# Make changes
# Write tests
# Update documentation

# Commit with conventional commits
git commit -m "feat: add amazing feature"

# Push and create PR
git push origin feature/your-feature
```
